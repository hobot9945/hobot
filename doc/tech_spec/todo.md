# Идеи/требования к “полному” инструментарию Hobot (GUI-исполнитель)

Ниже список команд/возможностей, которые стоит добавить или как минимум продумать, помимо уже сделанных
(мышь, фокус/поиск окон, скриншоты, базовая клавиатура).

---

## 1) Базовые “контрольные” команды (синхронизация/отладка)
- `sleep_ms` / `wait`  
  Явная пауза (в миллисекундах). Нужна для борьбы с гонками UI.
- `get_cursor_position`  
  Возвращает текущие координаты курсора (виртуальный рабочий стол). Полезно для отладки.
- `get_window_rect_by_hwnd`  
  Геометрия окна (left/top/right/bottom или x/y/width/height) по HWND.
- `get_window_rect_by_title`  
  Геометрия окна по needle (поиск + rect). Нужно для кликов относительно окна, а не “магических” координат.

---

## 2) Безопасность и “разлипание” ввода
- `press_escape` (или общий `key_press(VK_ESCAPE)`)  
  Универсально закрывает меню/всплывашки/модальные диалоги.
- `release_modifiers`  
  Принудительно отпускает Ctrl/Alt/Shift/Win (на случай “залипания” после ошибки/прерывания).
- Проверка фокуса перед вводом  
  Либо отдельная команда “убедиться что окно сфокусировано”, либо возвращаемый признак успеха из `focus_window_*`.

---

## 3) Текст: два режима ввода + clipboard
### 3.1 Clipboard-вставка
- `set_clipboard_text` (минимально нужно)  
  Положить текст в буфер обмена.
- `paste_clipboard` (Ctrl+V)  
  Вставка из буфера в текущий фокус.

### 3.2 Набор текста “как клавиатурой”
- `type_text`  
  Имитация ввода текста посимвольно/пакетно, не меняя clipboard.
  (Полезно там, где вставка запрещена, или важно не трогать буфер.)

### 3.3 Утилиты для полей ввода
- `select_all` (Ctrl+A)
- `delete_selection` (Backspace/Delete)
- Возможная составная команда: “очистить поле и вставить текст”.

---

## 4) Мышь: полезные расширения (если понадобится)
- `mouse_middle_click`  
  Часто нужно: открыть ссылку в новой вкладке, scroll-click.
- `mouse_move_by(dx, dy)`  
  Относительное движение, иногда удобнее абсолютного.
- (Уже есть/планируется) `mouse_scroll`, `mouse_drag`.

---

## 5) Клавиатура: примитивы + комбо
### 5.1 Примитивы
- `key_down(vk)`
- `key_up(vk)`
- `key_press(vk)`

### 5.2 Комбо
- `key_combo([modifiers...], key)`  
  Универсальная команда для Ctrl+S, Ctrl+Shift+T, Alt+F4 и т.д.

### 5.3 Готовые “часто нужные”
- `ctrl_c`, `ctrl_v`, `ctrl_a`
- `alt_f4`
- `enter`
- `tab`, `shift_tab`
- `ctrl_tab`, `ctrl_shift_tab`

---

## 6) Скриншоты: удобные режимы диагностики
- `capture_foreground_window`  
  Скрин активного окна (без необходимости знать needle/hwnd заранее).
- `capture_region_around_cursor(radius)`  
  Быстрый “прицельный” скрин вокруг курсора для отладки кликов/позиционирования.
- (Уже есть) наложение курсора + отчёт `Курсор: (...) [вне/внутри]`.
