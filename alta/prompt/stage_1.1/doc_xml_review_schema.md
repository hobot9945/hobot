# doc_xml_review_schema.md — Review этапа 1.1 (Формализация XML)

## Назначение

Файл `doc_xml_review.md` фиксирует результат генерации XML-документов на этапе 1.1 и выявленные технические проблемы.

Этап 1.1 считается завершенным только если:
- все обязательные документы сформированы;
- XML-файлы успешно записаны в `stage_1.1_result\<кейс>\formalized_docs\`;
- не осталось нерешённых блокеров (critical pending из primary.md).

- Даты приведены к формату `YYYY-MM-DD`
