# Шаблон отчета по этапу 2.0: Подготовка полей ДТ (dt_fields_review.md)

## Назначение

`dt_fields_review.md` — краткий отчет по результату построения `dt_fields.yaml`.
Нужен, чтобы оператор быстро понял:
- что получилось заполнить уверенно;
- какие поля остались `pending`;
- какие правила помечены как `птп` (правило требует подтверждения);
- что блокирует переход к этапу 2.1 (генерация XML ДТ).
