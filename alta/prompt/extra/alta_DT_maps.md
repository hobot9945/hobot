## Последовательность навигации к созданию новой ДТ в папке `СКИФ\ТМП`

### Базовые правила

- Перед любыми действиями с графическим интерфейсом сначала выполнять фокусировку окна Альты.
- Координаты мыши считаются относительно окна приложения Альта.
- Для клавиатурной навигации нужен фокус именно в центральной таблице.

### Приведение центральной таблицы в исходное состояние

1. Клик по пункту `Корзина` в боковой панели — `76:103`.
2. Клик по пункту `Основной каталог` в боковой панели — `90:86`.
3. Клик по строке `Корзина` в центральной таблице для установки клавиатурного фокуса.

Результат:

- активен корень `Основной каталог`;
- в центральной таблице выделена строка `Корзина`.

### Переход к папке `СКИФ`

1. Нажать `Down` **15 раз**.
2. Нажать `Enter`.

Результат:

- открыт каталог `СКИФ`;
- в центральной таблице появляется строка `Наверх`, затем `Запчасти для термометров`, `Сетки`, `ТМП`.

### Переход к папке `ТМП`

Из состояния после входа в `СКИФ`:

1. Нажать `Down` **3 раза**.
2. Нажать `Enter`.

Результат:

- открыт каталог `СКИФ\ТМП`.

### Создание новой ДТ

Находясь в папке `СКИФ\ТМП`:

1. Сфокусировать окно Альты.
2. Нажать `Ctrl+N`.

Результат:

- открывается новое окно документа вида  
  `ДТ (ГТД) 00000000/дата/номер`;
- отображается форма `ДЕКЛАРАЦИЯ НА ТОВАРЫ (ДТ)`.
