# doc_conversion_review_schema.md — Review этапа 0 (Конвертация первички)

## Назначение

Файл `doc_conversion_review.md` фиксирует результат конвертации первичных документов
(docx, xlsx, pdf, png и др.) в текстовый формат `.md`.

Цель этапа 0 — обеспечить удобную и стабильную текстовую базу для этапа 1.0.
Этап не проверяет бизнес-логику документов, только корректность извлечения текста.

---

## 1. Метаданные

- `название кейса`: <кейс>
- `путь к папке поставки`: <путь>
- `дата выполнения`: <YYYY-MM-DD>
- `режим`: <рабочий / отладка>
