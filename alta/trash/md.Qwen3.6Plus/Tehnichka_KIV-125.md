# ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ / SPECIFICATIONS  
**Продукт:** КИВ-125 — приточный клапан канального типа  
**Исх. №:** 1СК1004 от 10.04.2025  
**Производитель:** Ningbo ZENTEC Air Conditioning & Refrigeration Co., Ltd  

## Назначение  
Подача свежего воздуха в помещение с регулируемым потоком, фильтрацией, тепло- и шумоизоляцией.  
Установка: в стену, с наружной решёткой и внутренним оголовком с заслонкой и фильтром.  

## Основные параметры  
| Параметр | Значение |  
|----------|----------|  
| Тип | Приточный клапан канального типа |  
| Диаметр воздуховода | 125 мм |  
| Производительность | 30–60 м³/ч (в зависимости от перепада давления) |  
| Фильтрация | Фильтр грубой очистки, класс G3 |  
| Регулировка | Ручная заслонка |  
| Материалы | Оголовок: пластик ABS; изоляция: ППУ, ретикулированный поролон |  
| Размеры | 175 × 133 мм |  
| Рабочая температура | от -30 °C до +50 °C |  
| Код ТН ВЭД | 8481309908 |  
| Ставка пошлины | 7% |  
| Разрешительные документы | Не требуются |  

## Комплектация  
- Пластиковая труба с оголовком  
- Фильтр G3  
- Ручная регулировочная заслонка  
- Наружная защитная решётка
