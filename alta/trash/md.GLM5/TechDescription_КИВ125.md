## Описание товара

**КИВ-125** — приточный клапан, предназначенный для подачи свежего воздуха в помещение. 
Обеспечивает регулируемую подачу воздуха, фильтрацию и тепло- и шумоизоляцию. 
Внешне представляет собой пластиковую трубу с оголовком, устанавливаемую в стену.

## Основные характеристики

| Параметр | Значение |
|----------|----------|
| **Тип** | Приточный клапан канального типа |
| **Диаметр воздуховода** | 125 мм |
| **Производительность** | 30–60 м³/ч (в зависимости от перепада давления) |
| **Фильтрация** | Фильтр грубой очистки класса G3 |
| **Регулировка** | Подача воздуха регулируется заслонкой вручную |
| **Установка** | Монтируется в стену, снаружи — решётка, внутри — оголовок с заслонкой и фильтром |
| **Материалы** | Пластиковый оголовок ABS, ППУ, ретикулированный поролон |
| **Размеры** | 175×133 мм |
| **Диапазон рабочих температур** | от -30°C до +50°C |
